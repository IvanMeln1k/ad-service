# Notificator — API

Base URL: `/api/v1`

Notificator — универсальный транспорт для отправки уведомлений. Не знает ни про объявления, ни про пользователей. Получает готовое сообщение и доставляет его по указанному каналу.

## От любого внутреннего сервиса / воркера

| Метод | Путь                        | Описание                          |
|-------|-----------------------------|-----------------------------------|
| POST  | `/api/v1/notifications/email` | Отправить email по адресу получателя |

### `POST /api/v1/notifications/email`

**Request:**

```json
{
  "to": "user@example.com",
  "subject": "Тема письма",
  "body": "<p>HTML или текст письма</p>"
}
```

| Поле      | Тип       | Обязательное | Описание                     |
|-----------|-----------|:------------:|------------------------------|
| `to`      | EmailStr  | да           | Email получателя             |
| `subject` | string    | да           | Тема письма                  |
| `body`    | string    | да           | Тело письма (HTML или текст) |

**Response 200:**

```json
{ "status": "sent" }
```

**Ошибки:**

| Код | Описание                      |
|-----|-------------------------------|
| 422 | Невалидный запрос             |
| 502 | Ошибка отправки (SMTP недоступен) |
