# Profiler — API

Base URL: `/api/v1`

## От Registrator

| Метод  | Путь                                        | Описание                                      |
|--------|---------------------------------------------|-----------------------------------------------|
| POST   | `/api/v1/profile`                           | Создать профиль при регистрации               |
| DELETE | `/api/v1/profile/{user_id}`                 | Удалить профиль (компенсирующая транзакция)   |
| GET    | `/api/v1/profile/lookup?email={email}`      | Найти профиль по email → `[{user_id, name, avatar_url}]` |

## От RegistratorUnloader

| Метод | Путь                                               | Описание                              |
|-------|----------------------------------------------------|---------------------------------------|
| GET   | `/api/v1/profile/{user_id}/email-confirmation-token` | Получить токен подтверждения → `{token}` |

## От Browser (через Nginx)

| Метод | Путь                        | Описание                                    |
|-------|-----------------------------|---------------------------------------------|
| POST  | `/api/v1/email-confirmation` | Подтвердить email по токену (400 / 409 / 200) |

## От Adser

| Метод  | Путь                            | Описание                                     |
|--------|---------------------------------|----------------------------------------------|
| GET    | `/api/v1/profile/{user_id}`     | Получить профиль пользователя                |
| PATCH  | `/api/v1/profile/{user_id}`     | Обновить профиль (name, avatar, phone, city) |
| DELETE | `/api/v1/profile/{user_id}`     | Удалить аккаунт пользователя                 |
| GET    | `/api/v1/profile/{user_id}/roles` | Получить список ролей → `["MODERATOR"]`    |
| GET    | `/api/v1/profile`               | Список пользователей (только Admin)          |
